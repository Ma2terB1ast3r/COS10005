CSS Animations

Hover Animation

An animation can applied only while the element is being hovered over, which can be used to better show interaction with certain elements.

.growAnimation {
    width: 100px;
    height: 100px;
    background-color: var(--secondary);
    animation-name: none;
    margin: auto;
}

.growAnimation:hover {
    animation-name: growAnim;
    animation-duration: 2s;
    animation-iteration-count: infinite;
}

@keyframes growAnim {
    0%   {transform: scale(1);}
    50%  {transform: scale(1.25);}
    100% {transform: scale(1);}
}

Pulsing Animation

I made a pulsing animation that alternates between 2 colors which could be used to highlight important elements or elements that need attention.

.pulseAnimation {
    width: 100px;
    height: 100px;
    background-color: var(--secondary);
    animation-name: pulseAnim;
    animation-duration: 8s;
    animation-iteration-count: infinite;
    margin: auto;
}

@keyframes pulseAnim {
    0%   {background-color: var(--secondary);}
    50%  {background-color: var(--accent);}
    100% {background-color: var(--secondary);}
}

Spinning Animation

Spinning animations can be used for loading screens (if this site actually needed them).

.spinAnimation {
    width: 100px;
    height: 100px;
    background-color: var(--secondary);
    animation-name: spin;
    animation-timing-function: ease-in-out;
    animation-duration: 4s;
    animation-iteration-count: infinite;
    margin: auto;
}

@keyframes spin {
    0%   {transform: rotate(0deg);}
    100% {transform: rotate(360deg);}
}
